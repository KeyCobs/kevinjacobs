KevinJacobs.net (WIP)

This project is about my personal website. The one you are currently visiting. I've never been great at web development. But still wanted to try it out.

About the project:

I've taking advice from a YouTuber named NetworkChuck to create my own website. But I just didn't want to create a website from Wordpress. No I wanted to create one from scratch. I want to start with some basic HTML and CSS. I don't see any security flaws in this (There is no inlogging or anything) if anything this can just be a blog. Later on I want to create the site or update it useing Vue, React or angular. I wanted to showcase my work, my projects and I'll admit even brag a bit to my friends that I have my own personal website.

What I learned

There isn't a lot of things I've learned with this project. This project isn't something where I can learn from (even though I hope that I will learn something from this). It I do hope I can learn how to create a beautifull site. one that is fun to interact with. I'm planning to put in some fun easter eggs and even mini games where I can practice my JavaScript in.

What are the difficulties

I'm don't like developing website. I never did. The reason for this is because I can't create visual apealing sites, and because of this I can losse interest in it. I hope that this can change my way of thinking. Once I have the final product I can feel proud of what I've achieved
